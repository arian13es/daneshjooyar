# دانشجویار (Daneshjooyar) 🎓

**دانشجویار** یک اپلیکیشن جامع و پیشرفته‌ی دستیار دانشجویی است که برای کمک به برنامه‌ریزی، مدیریت وظایف، یکپارچگی با تقویم آموزشی و افزایش بهره‌وری دانشجویان طراحی شده است. این پروژه با استفاده از جدیدترین تکنولوژی‌های وب به صورت PWA توسعه یافته و قابلیت نصب به عنوان اپلیکیشن اندروید (با استفاده از Capacitor) را نیز دارد.

## 🌟 ویژگی‌های کلیدی و امکانات
بر اساس بررسی کدهای پروژه، امکانات زیر در اپلیکیشن گنجانده شده است:
- **تقویم و برنامه کلاسی (Schedule):** مدیریت برنامه هفتگی کلاس‌ها با رابط کاربری گرافیکی جذاب.
- **مدیریت امتحانات (Exams):** ثبت دقیق امتحانات، شمارش معکوس و ابزار **حالت تمرکز (Focus Mode)** برای مطالعه بهتر.
- **بورد پروژه‌ها و تکالیف (Project Board):** سیستم مدیریت تسک‌ها و پروژه‌های دانشگاهی.
- **رزرو سلف و غذا (Food Reservation):** بخش اختصاصی برای مدیریت برنامه غذایی دانشگاه.
- **دستیار هوشمند (AI Assistant):** بهره‌گیری از هوش مصنوعی گوگل (Google GenAI) برای ارائه دستیار هوشمند به دانشجویان.
- **نوتیفیکیشن‌های محلی:** یادآوری کلاس‌ها و امتحانات از طریق اعلان‌های گوشی (Capacitor Local Notifications).
- **تقویم کاملاً شمسی:** سازگاری کامل با تاریخ جلالی و مناسبت‌ها.

## 🛠️ تکنولوژی‌های استفاده شده
- **هسته و رابط کاربری:** React 19 + TypeScript
- **استایل‌دهی:** TailwindCSS v4 + Framer Motion (برای انیمیشن‌ها)
- **بیلد سیستم:** Vite
- **موبایل:** Capacitor (برای پورت اندروید/iOS و ارتباط با سخت‌افزار گوشی)
- **آیکون‌ها:** Lucide React
